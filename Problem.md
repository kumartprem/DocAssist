Document Collection Assistant (Start here)

This is the best first idea.

Why this is best
Every lawyer has this problem
Clients send documents incomplete and in parts
Lawyers waste a lot of time following up
Very easy to build MVP
Easy to show value quickly
Simple MVP idea

Build a small system like this:

Flow:

Lawyer creates a case
System generates checklist (ex: ID, medical records, forms)
Client gets link
Client uploads docs
System tracks:
Received
Missing
Auto reminders sent to client
What makes it powerful

Later you can add:

AI reads document and checks if correct
Auto detect missing info
Smart suggestions

But first version should be simple.
